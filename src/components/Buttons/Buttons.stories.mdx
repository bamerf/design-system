# Buttons

Buttons are used to trigger actions within an application.

## Primary Buttons

Primary buttons indicate the most important action on a page.

<Preview withToolbar>
	<Story name="Primary">
		<Button>Primary</Button>
	</Story>
</Preview>

## Secondary Buttons

Secondary buttons call for an action with less priority compared to Primary buttons.

<Preview withToolbar>
	<Story name="Secondary">
		<Button type="secondary">Secondary</Button>
	</Story>
</Preview>

## Tertiary Buttons

Tertiary buttons are used as opposites to Primary button. For example _Cancel_ against _Login_

<Preview withToolbar>
	<Story name="Tertiary">
		<Button type="tertiary">Tertiary</Button>
	</Story>
</Preview>

## Status Buttons

Status buttons are used to indicate the outcome of an action.

<Preview>
	<Story name="Success">

	<Story name="Warning">

	</Story>
	<Story name="Error">

	</Story>

## Sizes

You can choose between different sizes by specifying the `size` prop.

<Preview>
	<Story name="Small">

	</Story>
	<Story name="Medium">

	</Story>
	<Story name="Large">

	</Story>

## Icon Buttons

You can pass an Icon as children to the component. You need to include `type="icon"` for correct padding.

<Preview>
	<Story name="Icon Button">
		<Button type="icon">{DeleteIcon}</Button>
	</Story>
	<Story name="Labelled Icon Small">
		<Button type="labelledIcon" size="small">
			{DeleteIcon} Small
		</Button>
	</Story>
	<Story name="Labelled Icon">
		<Button type="labelledIcon">{DeleteIcon} Labelled</Button>
	</Story>
	<Story name="Labelled Icon Large">
		<Button type="labelledIcon" size="large">
			{DeleteIcon} Large
		</Button>
	</Story>
</Preview>
